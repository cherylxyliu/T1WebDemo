## 一句话概念

你在约会 App 上匹配到一个**已注销的用户**。你以为自己在去找 TA，实际上是 TA 正在沿着你的资料、距离和镜头，一步步走到你面前。

## 关键改动

- **写实大拇指常驻演出**：删除旧版气泡和 emoji 拇指，改用 CSS 拟物手指覆盖界面。
- **删除显性引导内容**：不再出现“拖这里/点这里/拇指这样做”的教学文案。
- **内心独白底图增强**：底部黑底白框字幕条更厚、更亮，与普通 UI 明显区分。
- **阶段二重做**：异常卡左滑弹回并显示指定独白；右滑后播放匹配动画，再进入聊天地址链接。
- **地图重做**：改成黑白高反差、颗粒、影印风格；拖拽推进后距离真实变化。
- **阶段三裁剪**：只保留跳一跳和擦拭路牌，删除墓碑拼图。
- **终局重做**：删除原死胡同和真实相机权限，改为伪系统权限弹窗 + 预渲染低光取景框。

## 阶段说明

### 阶段一：注册

- 输入昵称。
- 获取验证码并完成注册。
- 昵称会进入后续资料页、权限弹窗和注销结局。

### 阶段二：刷卡匹配

- 正常用户：金融男 / Livehouse 主理人 / 摄影师。
- 异常用户：全黑照片、昵称「已注销用户」、签名「来找我。别再左滑了。」
- 左滑异常卡：弹回原位，独白显示：`这个人还怪特别的，不如匹配一下试试...`
- 右滑异常卡：播放 Tinder 风格匹配动画。
- 匹配后进入聊天，对方发送地址链接：`来找我。`

### 阶段三：黑白地图与事件

- 地图采用 Finders Keepers 式视觉语言：黑白、高反差、噪点、照片卡片、写实手指覆盖。
- 每次有效拖拽底部按钮都会推进距离。
- 事件一：跳一跳，长按蓄力；成功显示`完美跳过水坑！`，失败显示`哎呀，踩到水坑里了...`。
- 事件二：擦拭路牌，滑动涂抹后显现`不要回头`，触发震动和强烈独白。

### 阶段四：资料更新与终局

- 擦拭路牌结束后通知：`【未知用户】的资料更新了，快去看看`。
- 点击后进入约会 App 资料页，未知用户签名和标签读取玩家行为。
- 距离在资料页中实时缩短至 0。
- 黑屏后模拟系统权限弹窗：`“玩家名称”想要访问你的相机`。
- 不论允许或拒绝，都进入伪前置摄像头画面；不调用真实相机。

## 技术规格

| 维度 | 规格 |
|------|------|
| 开发方式 | 纯 HTML + CSS + Vanilla JavaScript（单文件） |
| 依赖项 | 零外部依赖 |
| 屏幕比例 | 9:16 竖屏优先 |
| 触控事件 | touchstart / touchmove / touchend + mouse 兼容 |
| 相机 | 不调用真实相机；结局为伪权限弹窗 + 预渲染画面 |
| 震动 | `navigator.vibrate` 可用时用于擦拭路牌和终局提示 |
| 验证方式 | 打开 `test.html`，当前自动检查 11 个关键点 |
