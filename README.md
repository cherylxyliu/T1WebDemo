# 替身（The Replacement）

**当前版本：v1.3.0**

《替身》是一个偏心理惊悚的移动端 H5 Demo：玩家从普通约会匹配开始，逐步被引导进入 AR 导航、异常事件与“替身接管”的终局演出。

## 本版重点

- **写实手势演出**：删除旧引导气泡，改为常驻手势剪影与 AR 跟手拇指。
- **阶段三重做**：导航改成“上半 AR 实景 + 下半 2D 地图”，按住并上推可持续前进，距离会真实缩短。
- **路牌事件修正**：`不要回头` 默认不可见，必须擦拭到完成后才显现。
- **终局重做**：改为单按钮权限请求 → 伪造取景框识别 → 老电视息屏 → 伪造桌面 → 替身之手自动回复“妈妈”消息 → 黑屏标题落幅。
- **素材接入**：已把街景参考图和两张手部剪影放入 `assets/`，并在运行时自动抠掉浅色背景。

## 当前流程

1. 输入玩家名并开始匹配
2. 浏览 3 张正常资料卡
3. 第 4 张异常黑卡只能右滑，左滑会弹回并触发指定独白
4. 进入匹配动画与聊天页，对方发来地址链接“来找我”
5. 打开地图后进入 AR 导航：上推持续前进
6. 触发事件一：跳一跳（长按蓄力）
7. 触发事件二：擦拭路牌（擦完才显现 `不要回头`）
8. 路牌结束后收到资料更新通知，进入约会 App 外壳的异常资料页
9. 距离缩短到零，进入“替身接管”终局

- `reference/策划开发文档-v1.2.md`：上一轮归档保留

## 本地运行

直接启动静态服务器即可，例如：

```bash
python -m http.server 8123
```

然后访问 `http://127.0.0.1:8123/`。
